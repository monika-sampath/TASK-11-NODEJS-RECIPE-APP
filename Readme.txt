
Project Description: 


Developed a complete CRUD (Create, Read, Update, Delete) application for managing recipes using Node.js, Express.js, and Mongoose. The application will follow the MVC pattern and include comprehensive API documentation using Postman.


Functionalities used:


  The following functionality is achieved,


createRecipe: Create a new recipe.


getAllRecipes: Retrieve all recipes.


getRecipeById: Retrieve a single recipe by ID.


updateRecipe: Update a recipe by ID.


deleteRecipe: Delete a recipe by ID.



The MVC pattern with separate folders for models, views, and controllers is created.


Complete implementation of the CRUD operations is done.


Fully functional integration with MongoDB using Mongoose is done.


Detailed documentation for each API endpoint in Postman is given as json file.


Included sample requests and responses for each operation.


Used proper error handling and validation.


Clean, readable, and well-documented code.



Techstacks needs to be used : 


Node.js
Express.js
Mongoose (MongoDB)
Postman
